# Cyber Matrix Landing (React + Vite + Tailwind + Framer Motion)

Matrix-style landing page for technical services.

## Stack

- React
- Vite
- Tailwind CSS
- Framer Motion

## Development

1. Install dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

## Structure

- `src/components/MatrixBackground.jsx` — Matrix rain background on canvas (optimized for mobile).
- `src/components/Hero.jsx` — typewriter hero section and CTA.
- `src/components/Services.jsx` — terminal-style service cards with scroll animations.
- `src/components/Footer.jsx` — minimalistic footer with contacts.
- `src/App.jsx` — assembles everything, places content above fixed background.
